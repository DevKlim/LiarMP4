Detecting Contextual Malformation
with Fractal Chain-of-Thought

LiarMP4 benchmarks Predictive AI baselines against a novel Generative AI (GenAI) approach to content moderation. By recursively analyzing the semantic dissonance between visual evidence, audio waveforms, and textual claims, our multi-agent architecture catches sophisticated disinformation that traditional models miss.

Try the Live Demo Read the Report

Problem & Target Audience

Target User: Trust & Safety teams, journalists, and social media platforms.

The Problem: Traditional content moderation relies on Predictive AI (PredAI) evaluating metadata (e.g., account age, engagement velocity) to output a scalar probability. While efficient, this approach fails against Contextual Malformation: when a completely authentic video is paired with a fabricated caption. Deepfake detectors pass the video as "real," but the semantic intent is entirely deceptive.

We propose using Generative AI (GenAI) to extract distinct Veracity Vectors (Visual Integrity, Audio Integrity, Source Credibility, Logic, Emotion) to provide auditable and interpretable moderation signals.

Scope Boundaries

What We Built (Contributions):
  • Fractal Chain-of-Thought (FCoT) Orchestrator: A recursive inference strategy to verify modalities.
  • Human-In-The-Loop (HITL) Labeling Studio: A React-based UI to generate verified Ground Truth datasets.
  • TOON Parser: Strict Token-Oriented Object Notation parsing for reliable GenAI structured data output.
What We Reused:
  • Foundation Models: Gemini 2.0 Flash / Qwen3-VL for multimodal inference.
  • Google Agent Development Kit (ADK) for A2A communication.
  • AutoGluon/XGBoost for our Predictive AI baselines.
Out of Scope: Real-time live stream moderation (latency constraints) and generation of synthetic deepfakes.

Key Results & Evaluation

Models were evaluated against our manually verified Ground Truth dataset. We calculate a Composite Mean Absolute Error (MAE) measuring the absolute distance across 8 multi-dimensional Veracity Vectors, alongside overall Tag Accuracy.

Iteration / Method FCoT Depth Composite MAE ↓ Tag Accuracy ↑ Avg Latency
1. Baseline PredAI (AutoGluon/XGBoost) 0 38.4 N/A 1.2s
2. GenAI (Standard Chain-of-Thought) 1 24.1 68.2% 15.4s
3. GenAI (Fractal Chain-of-Thought) 2 12.8 84.5% 34.1s

How to Interpret Results

The Baseline PredAI acts as a fast filter but fundamentally fails to understand multimodal context (high error of 38.4). Implementing Deep FCoT (Iteration 3) reduces hallucinations significantly, driving the Composite MAE down by >50%. The Agent A2A architecture yields the highest accuracy but introduces significant latency tradeoffs.

Dataset & Rigor

Evaluation was conducted on a curated dataset built via our internal HITL studio. To prevent data leakage, the PredAI models were trained strictly on isolated tabular metadata from the training split, while GenAI models performed zero-shot semantic inference on the unseen test holdouts.

Methods & Architecture

LiarMP4 solves semantic evaluation by decoupling visual authenticity from textual intent.

Fractal Chain-of-Thought

A recursive three-stage prompt strategy. Macro-Scale forms a broad hypothesis of intent. Meso-Scale branches to verify specific pixel/audio anomalies. Synthesis forces the agent to cross-examine and self-correct if the Meso evidence contradicts the Macro hypothesis.

TOON Schema Enforcement

LLMs struggle to produce consistent large JSON blobs. We developed Token-Oriented Object Notation (TOON), a compressed schema that saves tokens and forces deterministic, tabular data output suitable for direct database integration.

Multi-Agent A2A Protocol

Built natively on Google's Agent Development Kit (ADK). The system dynamically shifts Prompt Personas and Base Models during live execution, utilizing centralized Evaluator agents to ground answers via Search Retrieval.

Limitations & Failure Modes

Inference Latency

Executing a Deep FCoT pipeline takes an average of 45.8 seconds per video. While highly accurate, this architecture cannot be deployed for real-time edge filtering without massive compute scaling.

"Text-Only" Hallucinations

When video downloads fail (due to DRM or API limits), the system falls back to text-only analysis of transcripts and captions. In this mode, the model is highly prone to over-categorizing political topics as "malicious" due to inherent foundation model safety biases.

API Cost / Token Limits

Recursive chain-of-thought consumes significantly more input/output tokens than standard scalar classification, making it cost-prohibitive for large-scale, unthrottled firehose ingestion.

Reproducibility & Deployment

The entire research pipeline—including the FastAPI backend, Go Server, React Frontend Studio, and Model dependencies—is open source and fully containerized via Docker.
