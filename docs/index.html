Detecting Contextual Malformation
with Fractal Chain of Thought

LiarMP4 benchmarks Predictive AI baselines against a novel Generative AI approach to content moderation. By recursively analyzing the semantic dissonance between visual evidence, audio waveforms, and textual claims, our multi-agent architecture catches sophisticated disinformation that traditional models miss.

Problem and Target Audience

Target User: Trust and Safety teams, investigative journalists, and social media platforms.

The Problem: Traditional content moderation relies on Predictive AI evaluating metadata such as account age and engagement velocity to output a scalar probability. While computationally efficient, this approach fails entirely against Contextual Malformation. This occurs when a completely authentic video is paired with a fabricated caption. Standard deepfake detectors pass the video as real, but the semantic intent is entirely deceptive.

We propose using Generative AI to extract distinct Veracity Vectors including Visual Integrity, Audio Integrity, Source Credibility, Logic, and Emotion to provide auditable and interpretable moderation signals.

Scope Boundaries

What We Built:
  • Fractal Chain of Thought Orchestrator: A recursive inference strategy to verify multiple modalities.
  • Human In The Loop Labeling Studio: A browser extension and UI to generate verified Ground Truth datasets.
  • TOON Parser: Strict Token Oriented Object Notation parsing for reliable structured data output.
What We Reused:
  • Foundation Models like Gemini 2.0 Flash and Qwen3-VL for multimodal inference.
  • Google Agent Development Kit for agentic communication.
  • AutoGluon and XGBoost for our Predictive AI baselines.
Out of Scope: Real-time live stream moderation due to latency constraints and generation of synthetic deepfakes.

Key Takeaways & Future Directions

  • FCoT Efficacy: Deep, recursive reasoning like depth 2 without external tools achieves the highest overall accuracy (64.8%) and effectively maps videos into a stable latent space, producing a robust Tagging Solution that prevents rigid, misclassified pigeonholing.
  • External Tools: Counterintuitively, providing the agent with Web Search and Code Execution decreased accuracy; however, it increased Tag Accuracy (from 64.8% to 92.1%), as the agent had better context on text results but weighed raw visual/audio alignment vectors less.
  • Credibility Tracking: Expanding our User Credibility Profiler, future versions will persistently track "Account Honesty", maintaining historical integrity scores that penalize accounts repeatedly posting recontextualized media.
  • Video-Post Alignment Database: We are establishing a dedicated database to track specific video-post alignment patterns, allowing the system to cross-reference known "cheap fakes" and flag recycled authentic videos weaponized with deceptive captions.

Ingestion and Manual Labeling

Our dataset originates directly from live platforms like Twitter and Reddit. We built a custom browser extension that extracts the video file, caption text, and metadata straight from the interface.

This raw data feeds into our batch ingestion queue. Human annotators use our custom labeling studio to review the queue and assign scores across distinct veracity vectors. This manual review establishes our Ground Truth dataset.

Human Validation

Annotators strictly follow the Veracity Rubric to establish mathematical baselines before exposing the data to the Generative AI agents.

Tagging and Topic Clustering

The Multi-Agent system assigns strict tags to every ingested video using Token Oriented Object Notation. By assigning tags such as Political, Satire, or Sensationalist, we map posts into a multi-dimensional latent space.

We then evaluate the spatial distance between different topics. When the agent uses multimodal reasoning, the distance between ostensibly different topics converges dynamically. A satirical political video might structurally mimic an entertainment sketch, pulling it closer to entertainment in the contextual vector space. This prevents the rigid misclassified pigeonholing rampant in traditional metadata-focused systems.

Work in Progress

Community Context and Comments

Social media platforms are highly reactive ecosystems. The LiarMP4 architecture is expanding to ingest the immediate community reaction to establish baseline sentiment prior to running forensic analysis.

Crowdsourced Grounding

The browser extension scrapes user replies to dynamically capture community sentiment. A specialized Context Agent summarizes these disparate replies into a concise consensus report.

This provides the Verification Agent with valuable skepticism and crowd-sourced fact checking before it begins processing the raw video pixels. This actively grounds the hallucination parameters of the Generative AI.

Active Development

This feature is currently undergoing refinement to better handle spam filters, bot networks, and malicious brigading within comment threads.

Multi-Agent Architecture and Fractal Logic

LiarMP4 solves semantic evaluation by decoupling visual authenticity from textual intent. We utilize a unique Fractal Chain of Thought process managed by distinct specialized agents.

Orchestrator Agent

Ingests context and forms a Macro-Scale intent hypothesis.

Vision & Audio Agents

Meso-Scale extraction of pixel and waveform anomalies.

Context Agent

Scrapes community comments to ground the assessment.

Verification Agent

Forces cross-examination. Self-corrects to output TOON Veracity Vectors.

1. Macro-Scale

The model ingests the entire context to form a broad hypothesis of intent. It attempts to answer whether the content feels like political satire, hard news, or malicious bait before investigating the pixels.

2. Meso-Scale

The pipeline branches to verify specific anomalies. It forces the agent to stop looking at the big picture and strictly analyze lip-sync synchronization, compression artifacts, and distinct logical fallacies.

3. Synthesis

The system forces the agent to cross-examine itself. If the Meso-Scale evidence contradicts the Macro-Scale hypothesis, the agent must abandon its initial assumption and format the truth into strict Token Oriented Object Notation.

Development Timeline and Iteration

Our approach evolved significantly as we identified the limitations of standard vision models against sophisticated real-world misinformation.

Phase 1: Open Source Vision

Our initial exploration utilized solely open-source Vision Language models such as Qwen3-VL. The primary goal was detecting spatial anomalies and deepfakes. We quickly discovered that high visual accuracy was insufficient for solving modern misinformation because genuine footage is routinely weaponized with false text.

Phase 2: Factuality Alignment

We integrated the Professor Ali Arsanjani's Factuality Factors (visit [Alternus Vera](https://alternusvera.wordpress.com/)) and Modality Alignment criteria. This shift allowed us to evaluate how audio, video, and text relate to one another. This successfully penalized malicious content where authentic videos were recontextualized with deceptive claims.

Phase 3: Multi-Agent System

To handle the complexity of processing isolated factuality vectors, we built a highly scalable architecture natively around the Google Agent Development Kit. This enables recursive verification steps and dynamic community context integration.

Limitations and Failure Modes

Inference Latency

Executing a deep pipeline takes an average of 45 seconds per video. While highly accurate, this architecture cannot be deployed for real-time edge filtering without massive compute scaling.

Text-Only Hallucinations

When video downloads fail due to DRM limits, the system falls back to text-only analysis of transcripts and captions. In this mode, the model is highly prone to over-categorizing political topics as malicious due to inherent safety biases.

API Cost

Recursive logic consumes significantly more input and output tokens than standard scalar classification, making it cost-prohibitive for large-scale unthrottled firehose ingestion.
